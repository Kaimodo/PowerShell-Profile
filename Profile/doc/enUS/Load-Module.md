# Load-Module

Install and Import Module.

## Syntax

```PowerShell
Load-Module
    -Name <String>
    -ArgumentList <String>
    [<CommonParameters>]
```

## Description

Import the specific Module. if Its not Installed, do so and import it.

## Examples

### Example 1:

./Load-Module PSWindowsUpdate
